# Site Provedor - AlfaTech

Landing page de um provedor fictício de hospedagem, com página principal e tabela de preços.

## Preview
<img width="1856" height="847" alt="image" src="https://github.com/user-attachments/assets/5b360c12-e4c7-4e34-b773-54bed679bff3" />


## Funcionalidades
- Página inicial com seções de informações, planos e contato
- Página dedicada de tabela de preços
- Navegação entre páginas (`index.html` e `tabela.html`)
- Layout com foco em apresentação comercial

## Tecnologias
- HTML5
- CSS3
- Google Fonts (Poppins)
